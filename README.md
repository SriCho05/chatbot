


## Animal Husbandry Department Chatbot

This project is an internal chatbot solution for the Department of Animal Husbandry. It provides automated FAQ and grievance support to users via WhatsApp, using an Excel-based FAQ sheet and AI-powered answers.

**Note:** This is a private, company-owned project. All rights reserved.




## Installation

1. Install Node.js v18 or newer on your system.
2. Clone this repository.
3. Run `npm install` to install dependencies.
4. Start the bot with `node bot.js`.




## Example usage

See `bot.js` for the main application logic and usage instructions. The bot will prompt for WhatsApp QR code authentication and guide users through language selection, FAQ, and grievance options.
